Pengertian Turunan Fungsi Aljabar

Turunan adalah pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai yang dimasukkan, proses dalam menemukan turunan disebut diferensiasi. Turunan fungsi aljabar adalah fungsi yang menurunkan pangkat dari fungsi sebelumnya sesuai aturan yang sudah ditetapkan. Biasanya, turunan aljabar tidak hanya bisa menurunkan dalam satu tingkat saja, tetapi bisa menurunkan dua, tiga, empat tingkat dan seterusnya.

Rumus-Rumus Turunan Fungsi Aljabar

Contoh Turunan Fungsi

Misalkan kita memiliki fungsi f(x) = 2x^2 + 3x. Turunan pertama dari fungsi ini adalah f'(x) = 4x + 3. Ini menunjukkan bagaimana fungsi f(x) berubah seiring perubahan nilai x.

Aturan-Aturan Turunan Aljabar

Berikut adalah beberapa aturan dasar dalam turunan fungsi aljabar:

1. Aturan Turunan Pangkat

Jika f(x) = x^n, maka turunannya adalah: f'(x) = n*x^(n-1).

Contoh: f(x) = x^3 memiliki turunan f'(x) = 3x^2.

2. Aturan Jumlah

Jika f(x) = g(x) + h(x), maka turunannya adalah: f'(x) = g'(x) + h'(x).

Contoh: Jika f(x) = 3x^2 + 5x, maka turunannya adalah: f'(x) = 6x + 5.

3. Aturan Perkalian

Jika f(x) = g(x) * h(x), maka turunannya adalah: f'(x) = g'(x) * h(x) + g(x) * h'(x).

Contoh: Jika f(x) = x^2 * (3x + 2), maka turunannya adalah: f'(x) = 2x * (3x + 2) + x^2 * 3 = 6x^2 + 4x + 3x^2 = 9x^2 + 4x.

4. Aturan Pembagian

Jika f(x) = g(x) / h(x), maka turunannya adalah: f'(x) = (g'(x) * h(x) - g(x) * h'(x)) / (h(x))^2.

Contoh: Jika f(x) = (x^2 + 1) / (x - 1), maka turunannya adalah: f'(x) = ((2x) * (x - 1) - (x^2 + 1) * 1) / (x - 1)^2 = (2x^2 - 2x - x^2 - 1) / (x - 1)^2 = (x^2 - 2x - 1) / (x - 1)^2.

5. Aturan Rantai

Jika f(x) = g(h(x)), maka turunannya adalah: f'(x) = g'(h(x)) * h'(x).

Contoh: Jika f(x) = (3x + 2)^4, maka turunannya adalah: f'(x) = 4(3x + 2)^3 * 3 = 12(3x + 2)^3.

Contoh Soal

1. Turunan pertama dari fungsi f(x) = 3x^4 + 2x^2 - 5x adalah ...

Jawaban:

f(x) = ax^n ⇒ f'(x) = a.n.x^n-1

f(x) = 3x^4 + 2x^2 - 5x

f'(x) = 4.3x^4-1 + 2.2x^2-1 - 1.5x^1-1

f'(x) = 12x^3 + 4x^1 - 5x^0

f'(x) = 12x^3 + 4x - 5

2. Turunan dari f(x) = x^3 adalah .

Jawaban:

f(x) = x^n ⇒ f'(x) = n.x^n-1

f(x) = x^3

f'(x) = 3x^3-1

f'(x) = 3x^2

3. Carilah turunan f'(x) dari fungsi f(x) = 3x^2 + 7x!

Jawaban:

f(x) = ax^n ⇒ f'(x) = a.n.x^n-1

f'(x) = 2.3x^2-1 + 1.7x^1-1

f'(x) = 6x^2 + 7x^0

f'(x) = 6x^2 + 7

4. Carilah turunan dari y = (2x + 1) 2x^2!

Jawaban:

y = U . V ⇒ y' = UV' + U'V

U = 2x + 1

U' = 2

V = 2x^2

V' = 2.2x2-1

= 4x^1

= 4x

y' = (2x - 1)(4x) + (2)(2x^2)

y' = 8x^2 - 4x + 4x^2

y' = 12x^2 - 4x

5. Turunan kedua dari fungsi aljabar f(x) = x^3 - 2x^2 + 3x adalah ...

Jawaban:

f'(x) = 3x^3-1 - 2.2x^2-1 + 1.3x^1-1

f'(x) = 3x^2 - 4x^1 + 3x^0

f'(x) = 3x^2 - 4x + 3

f''(x) = 2.3x^2-1 - 1.4x^1-1 + 0

f''(x) = 6x^1 - 4x^0

f''(x) = 6x - 4

Kesimpulan

Turunan membantu kita untuk memahami seberapa cepat suatu fungsi berubah. Ini adalah konsep dasar dalam kalkulus yang banyak digunakan dalam berbagai bidang ilmu, seperti fisika dan ekonomi.
